Intro to Markdown

  • Complete the Learning Lab Course - Communicating using Markdown
  • Show us some awesome stuff about you using Markdown. Let your imagination run wild!
  • Post a message on ATFG Discord #students channel with the hashtag #markdown (see example message below)
  • fun-fact: chit chat in the #students channel can earn you karma points.

Example Message

Hi! I've completed the Communicating using Markdown course.
My repository URL is https://github.com/rajeev/communicating-using-markdown and my website URL is rajeev.github.io/communicating-using-markdown
#markdown

(You can also post your own custom message with your repository URL and the hashtag #markdown)
